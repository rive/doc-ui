# @rive/react-doc-ui

Playground UI to wrap React Live.

- [ ] Live code editing
- [ ] Mobile, tablet, desktop preview switch

## Install

```bash
npm install --save @rive/react-doc-ui
```

## Usage

```jsx
import { Playground } from '@rive/react-doc-ui';

<Playground code="<div>Hello, World!</div>" />;
```
