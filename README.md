# @rive/doc-ui

React components to render document.

- [ ] React live playground
- [ ] Mobile, tablet, desktop preview switch

## Install

```bash
npm install --save @rive/doc-ui
```

## Usage

```jsx
import { Playground } from '@rive/doc-ui';

<Playground code="<div>Hello, World!</div>" />;
```
